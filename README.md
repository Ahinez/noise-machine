# 🌧️ Noise Machine

Background noise app for focus. No sign-up, no settings, no clutter.

**[Live Demo](https://noise-machine-eight.vercel.app/)**

![demo](./demo.gif)

## Features

- 5 ambient sounds: rain, café, forest, white noise, thunderstorm
- Mix multiple sounds simultaneously
- Per-sound volume control
- Real-time waveform visualization via Web Audio API + Canvas

## Stack

React 19 · TypeScript · Web Audio API · Canvas API · Vite

## Run locally

```bash
git clone https://github.com/Ahinez/noise-machine
cd noise-machine
npm install
npm run dev
```
